# vsc-base

vsc-base as a module that aims to make it esaier to create vscode extensions.

It also the base for vsc-script extension.

Documentation can be found on [vsc-base.org](http://vsc-base.org)

## Get started

```
>> npm i vsc-base
```

or

```
>> yarn add vsc-base
```
